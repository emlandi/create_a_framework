Create a Framework

The purpose of this assignment is to create a series of help libraries/functions for creating REST APIs using the node HTTP module. Some of the functionalities that we included are:

  • a 404 function
  • a writeHead function that includes status and content type
  • a writeSync function to synchronously add files based on the directory

Base Styles

We also incorporated the Base Styles assignment here. Specifically, we used:

  • Normalize.css as a CSS reset
  • 'Permanent Marker' and 'Arvo' from Google Fonts for typography
  • the Carolina palette from Adobe Color for color inspiration
  • Github icon from icomoon
