Welcome to My Page

The purpose of this assignment is to build an HTTP server with different routes, as follows:

  • Respond to a get request to '/time' that will send back the current time
  • Respond to a post request to '/greet' that will send back a greeting.
  • Respond to a get request to '/greet/name' (where name is any single word string) and send back a string that greets that name.
  • Sends back a 404 to any other requests.
